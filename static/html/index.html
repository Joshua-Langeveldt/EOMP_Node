Welcome To AniVerse

Route Method Description
/user GET Fetches a list of all users.
/user/:id GET Retrieves details of a specific user by ID.
/user/register POST Registers a new user.
/user/:id PATCH Updates user information (e.g., profile edits).
/user/:id DELETE Deletes a user account.
/user/login POST Handles user login (authentication).
/product GET Fetches a list of all products.
/product/recent GET Retrieves recently added products.
/product/:id GET Retrieves details of a specific product by ID.
/product/add POST Adds a new product to the database.
/product/:id PATCH Updates product information (e.g., price change).
/product/:id DELETE Deletes a product from the database.
